
# WRF-Hopper-Documentation

A comprehensive guide for compiling and running the Weather Research and Forecasting (WRF) model on George Mason University's Hopper HPC cluster.

## Overview

This repository provides step-by-step instructions and tutorials for working with the Weather Research and Forecasting (WRF) and WRF Preprocessing System (WPS) models on GMU's Hopper High-Performance Computing cluster. Whether you're new to WRF or looking for Hopper-specific configuration details, you'll find documentation here to help you get started.

## Repository Contents

### 📋 [WRF_WPS_Compile_Instructions_Hopper.md](WRF_WPS_Compile_Instructions_Hopper.md)
Complete step-by-step instructions for compiling WRF (em_real) and WPS on the Hopper cluster using the GNU compiler and OpenMPI toolchain. Includes:
- Module loading and environment setup
- NetCDF configuration
- Compilation procedures
- Troubleshooting tips

### 🎓 [matthew_tutorial/](matthew_tutorial/)
A practical tutorial for running a real-world simulation case (Hurricane Matthew, October 2016) on Hopper. This tutorial walks through:
- WPS preprocessing steps
- WRF model configuration and execution
- Data management and output handling
- A complete worked example from start to finish
